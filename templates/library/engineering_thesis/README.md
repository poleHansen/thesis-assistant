# Engineering Thesis Template

`word/template.docx` is a real placeholder-based Word template.

- The cover title is filled from `{{cover.题目}}`.
- The thesis body is filled with first-level section placeholders like `{{section.摘要}}` and `{{section.第1章 绪论}}`.
- The template keeps only first-level headings so the generated thesis content is inserted in place instead of being appended at the end.

`ppt/template.pptx` can still be replaced with a custom defense slide template when needed.
